# Team-11-ML-end-sem-project-
**Project Overview**

This project presents an **Adaptive Edge-Aware Anomaly Detection Framework** designed for IoT-based healthcare systems. With the rapid adoption of IoT medical devices for remote monitoring and real-time diagnostics, ensuring secure and efficient data processing has become a critical challenge.

Traditional intrusion detection systems rely on centralized machine learning models, which introduce high latency, increased bandwidth usage, and limited adaptability to resource-constrained edge devices. These limitations can negatively impact real-time healthcare applications where timely and accurate responses are essential.

To address these challenges, this project proposes a novel approach that combines **edge computing with dynamic feature compression**. The system intelligently adjusts the number of features used for anomaly detection based on the computational capacity of each IoT device. This enables efficient processing on low-power devices while maintaining strong detection performance.

The proposed framework works by:
- Processing network traffic data generated by IoT healthcare devices  
- Dynamically reducing feature dimensionality based on device constraints  
- Deploying lightweight machine learning models at the edge  
- Detecting anomalies such as cyber-attacks (e.g., spoofing, DDoS, data manipulation) in real time  

By integrating adaptive feature compression with edge-aware learning, the system achieves:
- Reduced latency for faster decision-making  
- Lower bandwidth consumption due to compressed data transmission  
- Improved scalability across heterogeneous IoT devices  
- Comparable or improved detection accuracy compared to traditional centralized approaches
